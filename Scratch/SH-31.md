SH-31

Found and fixed a bug where "No results" school data was being overridden a default set of schools that display when no results are found.  This caused the default data set to be interpreted as the last known results when the page was refreshed.  It was necessary to fix this bug to complete the current task.  Also had to adjust how the no-results signal was being interpreted when the page was reloaded, after a no-results search

The link structure includes a checksum of the original results document.  The checksum is verified against retrieved results record before returning data.  This prevents someone from potentially pulling up random records by playing with the URL.
